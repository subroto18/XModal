# XDisplayName

XDisplayName is a simple React application that takes a user's **first name** and **last name** as input and displays the **full name** dynamically on the screen.

## 🚀 Features

- Accepts **First Name** and **Last Name** from the user
- Combines both inputs to display the **Full Name**
- Built with **React**
- Clean and simple UI
- Demonstrates **form handling and state management**

## 🛠️ Tech Stack

- React
- TypeScript
- CSS

## ⚙️ How It Works

1. The user enters their **First Name** and **Last Name** in the input fields.
2. React state stores the values entered by the user.
3. The application combines both values.
4. The **Full Name** is displayed dynamically.

Example:

```
First Name: John
Last Name: Doe

Output:
Full Name: John Doe
```

## 📦 Installation

Clone the repository:

```
git clone https://github.com/subroto18/XDisplayName.git
```

Navigate to the project directory:

```
cd XDisplayName
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
