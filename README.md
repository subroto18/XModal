# 🧮 React Calculator App

A simple **basic calculator application** built using **React.js**.  
This project performs standard arithmetic operations like addition, subtraction, multiplication, and division with a clean and responsive UI.

The goal of this project was to practice **React fundamentals, state management, event handling, and UI layout design**.

---

## 🚀 Features

- Perform basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (\*)
  - Division (/)
- Real-time input display
- Clear / reset functionality
- Responsive calculator layout
- Clean and simple UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
